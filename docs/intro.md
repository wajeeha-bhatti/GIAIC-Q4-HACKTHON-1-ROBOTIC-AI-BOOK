# Tutorial Introduction

Let's get started with **creating your Physical AI book site** in just a few minutes.

## Getting Started

You can start by **setting up a new Physical AI book project**.

Or **try it immediately** with an online starter template for documentation sites.

### What You'll Need

- [Node.js](https://nodejs.org/en/download/) version 20.0 or above  
  - When installing Node.js, it is recommended to select all checkboxes related to dependencies.

## Generate a New Book Project

Create a new book project using the **classic Docusaurus template**:

```bash
npm init docusaurus@latest physical-ai-book classic
