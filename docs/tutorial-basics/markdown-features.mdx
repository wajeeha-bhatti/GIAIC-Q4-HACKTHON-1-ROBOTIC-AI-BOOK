# Markdown Features

Your Physical AI book supports **Markdown** with some additional features to enhance readability, include images, code, and interactive content.

## Front Matter

Markdown documents can include metadata at the top called **Front Matter**. This helps define the chapter, title, description, and custom URLs:

```text title="chapter-intro.md"
---
id: chapter-intro
title: Introduction to Physical AI
description: Key concepts of embodied intelligence and Physical AI systems
slug: /introduction
---
