# Create a Document

In this book framework, documents are **groups of pages** connected through:

- a **sidebar** for easy navigation  
- **previous/next links** between chapters  
- **versioning or updates** for different editions  

## Create your first Doc

Create a Markdown file at `docs/introduction.md`:

```md title="docs/introduction.md"
# Introduction to Physical AI

This is the **first chapter of your Physical AI book**!  
It introduces the key concepts of **embodiment**, **physical intelligence**, and how AI can act in the real world.
